# 工作台交互说明（简版）

## 1. 画布与节点
- 拖拽左侧节点模板到画布生成节点。
- 选中节点后可在右侧详情面板编辑 `name/action/targets/with`。
- 支持一键删除节点与自动布局。
- 自动保存草稿：节点/布局变化会触发草稿保存（graph + YAML）。

## 2. 聊天抽屉
- 右侧固定贴边，可折叠并持久化。
- 入口：
  - 生成流程
  - 修复流程
  - 选中节点时的“重生成节点”
- 生成/修复/重生成后自动同步画布。

## 3. 校验与自动修复
- 点击“校验”触发自动修复循环（流式日志）。
- 修复完成后会刷新画布并展示差异/问题摘要。

## 4. 运行执行
- 点击“运行”将当前 YAML 发送至运行 API。
- SSE 事件驱动运行日志与摘要面板更新。
- 结束后展示成功/失败统计、耗时与问题列表。

## 5. 图与 YAML 冲突
- 当加载草稿时检测 YAML 与图不一致。
- 弹窗提示用户选择“以 YAML 覆盖图”或“以图覆盖 YAML”。
