## 端到端流程
- [ ] 生成：在聊天抽屉输入需求，点击生成，画布生成节点并自动布局。
- [ ] 草稿：生成后刷新页面，草稿仍可加载（graph + YAML）。
- [ ] 拖拽：从节点库拖拽一个模板到画布，节点落位并自动保存。
- [ ] 编辑：选中节点并修改 `name/action/with/targets`，画布与草稿同步。
- [ ] 重生成：选中节点，输入意图重生成，节点内容更新。
- [ ] 校验：点击“校验”，接收流式状态，出现修复差异/摘要。
- [ ] 修复：若有问题，确认 YAML 被修复并重新生成图。
- [ ] 运行：点击“运行”，日志面板收到 step 事件与结束摘要。
- [ ] 摘要：运行结束后显示成功/失败统计、耗时、问题列表。

## 补充验证
- [ ] 抽屉折叠：折叠/展开后刷新仍保持状态。
- [ ] 冲突提示：手动篡改草稿 YAML 后重新加载，出现“图/YAML 冲突”提示。
- [ ] 失败兜底：关闭 AI 服务时，生成/修复提示错误且不崩溃。
